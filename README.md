🌐 Proyecto Final – Desarrollo Web (Coderhouse)

Este proyecto corresponde a la entrega final del curso de Desarrollo Web de Coderhouse. Consiste en la creación de un sitio web completo, aplicando buenas prácticas de maquetación, estilos, responsive design, SEO y despliegue.

📌 Objetivo del Proyecto

Desarrollar un sitio web funcional y visualmente atractivo, aplicando todos los conocimientos adquiridos durante el curso:

Estructura HTML semántica
Estilización con CSS y SASS
Diseño responsive (Mobile First)
Uso de Grid y Flexbox
Implementación de framework (Bootstrap)
Aplicación de SEO
Animaciones e interactividad
Publicación en servidor
🧩 Estructura del Sitio

El sitio cuenta con 5 secciones principales, desarrolladas en distintos archivos HTML:

index.html (página principal)
Otras páginas internas con navegación interconectada

Cada página incluye:

Estructura HTML clara y semántica
Enlaces a hojas de estilo y librerías externas
Contenido multimedia (imágenes optimizadas)
Navegación consistente
